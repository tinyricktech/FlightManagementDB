Flight Management Database
A command-line Python application for managing airline flights, pilots, and destinations using SQLite.
Requirements
- Python 3.x
- sqlite3 (included with Python)
How to run
1. Clone the repository
2. Open in GitHub CodeSpaces or run locally
3. Run python main.py
Features
- Add, view, update and delete flights
- Assign pilots to flights
- View pilot schedules
- Manage destination information
