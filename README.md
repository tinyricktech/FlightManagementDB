Flight Management Database
A command-line Python application for managing airline flights, pilots, and destinations using SQLite.
Requirements
1.	Python 3.x
2.	sqlite3 (included with Python)
How to run
1.	Clone the repository
2.	Open in GitHub CodeSpaces or run locally
3.	Run python main.py
4.	Select option 9 to load sample data
Features
1.	Add, view, update and delete flights
2.	Assign pilots to flights
3.	View pilot schedules
4.	Manage destination information
